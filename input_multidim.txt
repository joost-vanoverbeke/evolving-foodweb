
parameters are tagged with capital words
lines to be read as input should start with the parameter tag followed by the input value
all other text is ignored
lines not starting with an input tag (e.g. starting with '//') are ignored
non initialized parameters receive a default value

ecological parameters
ENVDIMS		        1											number of niche dimensions (default = 1)
TRAITS		        1											number of traits (default = 2)
MINENV              -0.05                                      	minimum environmental value (between 0 and 1) (default = 0.2)
MAXENV              0.05                                      	maximum environmental value (between 0 and 1) (default = 0.8)
MICROSITES          10000    										number of microsites in each patch (default = 600)
INRATE              200
NBRSPECIES          15
TROPHICLEVELS       3
D                   0.1                                      	mortality rate (default = 0.1)
REPTYPE             ASEXUAL											initial probability of sexual reproduction (default = 0)
GRIDX               5
TORUSX              NO
GRIDY               1
TORUSY              YES
ENVTYPE				LOCAL
PCHANGE		        0.1											probability of environmental change along each niche dimension (default = 0.1)
ENVSTEP		    1   0.0										    step size of environmental change (default = 0.01)
M               1   0.01		                     			    dispersal rate between patches (should be set to 0 if gridsize = 1 (one patch)) (default = 0.01)
RHO                 1                                        	scaling factor determining the width of the dispersal kernel  (default = 1)

evolution parameters
OMEGAE              0.5                                     	niche tolerance (default = 0.02)
TRAITLOCI           20                                       	total number of diploid loci coding for traits (default = 20 => 10 for each of 2 default trait)
MU                  1e-4                                     	mutation rate (default = 1e-4)
SIGMAZ              0.01                                     	phenotypic standard deviation (default = 0.01)


run parameters
RUNS                1                                        	number of replicate runs (default = 1)
TIMESTEPS           1                                    	number of time steps in a run (default = 10000)
PRINTSTEPS          1		                               	time steps between console output (default = 100)
SAVESTEPS           1                                   	time steps between generation of output (default = 1000)
OUTPUT              results_test.csv			  destination of output file (default = output_evolvingMetacommunity.csv)
